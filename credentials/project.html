CrypShare

CrypShare is a secure LAN file-sharing tool I made using Python and PowerShell, built to let users upload/download files over a local network via browser with HTTPS support (self-signed or mkcert). It includes features like password protection, QR-code access, automatic file watching & live UI refresh, and a slick neon-dark interface. This project shows my skills in secure networking, UI/web integration, and automation in system/tool development.

CrypShare - V2

CrypShare V2 is an upgraded local-network file sharing and streaming app I built using Python. It allows users to upload, download, and even stream videos (like anime or movies) directly through the browser without any external server. The system runs entirely on LAN, featuring password protection, QR-code access, and real-time file list updates. Designed for simplicity and privacy, CrypShare V2 delivers fast, seamless media access within a clean and responsive web interface - showcasing my skills in backend logic, file streaming, and full-stack web integration.

Universal File Converter

Universal File Converter is a Flask-based web app I built to convert images, audio, and video files into different formats directly in the browser. It supports batch uploads, real-time progress tracking, and automatic ZIP downloads for converted results. The backend handles file processing using FFmpeg, Pillow, and OpenCV, while the frontend is a clean, responsive interface built with HTML, CSS, and JavaScript. This project showcases my full-stack skills in multimedia processing, server-side automation, and interactive UI design.
